## 💡 التمهيد: إزاي تفهم شرحي؟

1. **الشكل المختصر (`//`):** شرح سريع في سطر واحد.
2. **الشكل الكامل (`/* ... */`):** "البرواز" اللي بحكي فيه قصة المسألة وفكرتها.
